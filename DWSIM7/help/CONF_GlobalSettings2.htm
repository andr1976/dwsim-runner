Global Settings

This tool is used to edit global settings of DWSIM

User-Defined Compound Databases


This table lists all user-defined compound databases being used in DWSIM. During startup of DWSIM all components of these databases are loaded.

Symbols


Disconnect: This will remove the database from list.

Edit: Opens Database Manager to edit this database.

Tasks


Add User Database: You can add a user database with your own components. User databases contain user defined components which were created by the Component creator utility.

Replace components option: If this checkbox is checked, user components will replace existing components if the component ID is identical.
