Global Settings

This tool is used to edit global settings of DWSIM

Backups


Backup Copies

This option causes DWSIM to create a cyclic backup copy. You also can select a folder where to save this backups and define a interval for cyclic savings.

In cases where a problem occures or DWSIM is forced to stop, this backup will prevent that all your work is lost. Your latest backup is recognised and you will be asked on startup to either load your backup copy or start regularly.

'Keep a copy' option
Check this box to save a backup copy when overwriting an existing file. The old file will have a "_backup" suffix added.
