UNIFAC Interaction Parameter Regression

Data & Regression

This page is used to enter experimental data, run the regression and display results.
At the moment this tool only can be used with LLE data for regression.

Data table

You may copy experimental data from other programs and paste them into this table. To delete a complete row you press Strg+Del.

Columns X1'/X1":These both columns contain molar fractions of component 1 (here Benzonitril) in liquid phase 1 ( X1') and liquid phase 2 (X1").
Column T: Enter temperature of every dataset here.

The following columns contain results and can not be edited.

As soon as you delivered all experimental data you may press button
Regress IPs : Regression of the selected interaction parameters are done. The final results are written to the parameter input fields on page "Model".
The regression procedure is as following for Modified UNIFAC models: For every data set a combination of "aij" and "aji" parameters is regressed. These parameters are displayed in the green columns. After finishing that procedure for every data set an interpolation over temperature is done to calculate final parameters for aij/aji and bij/bji which represent temperature dependence of interaction parameters.

If the selected model is UNIFAC the calculations are done in a slightly different way. This model is not able to account for temperature dependence of interactions and a reasonable parameter set has to be found. DWSIM selects the middle of the experimental temperature range to calculate interaction parameters from interpolated values. With the button
New Temperature you also can recalculate interaction parameters for a different and more suitable temperature. This button together with a temperature input field is only visible when using UNIFAC as model.

Draw Diagram: This button will draw a diagram of experimental data together with calculated data from actual interaction parameters. You may also draw this diagram in advance to regression or after setting a new reference temperature for UNIFAC model to compare the changes. The calculated equilibrium data are displayed in the grey columns together with an error indicator of that dataset.

Draw IPs: This button wil draw a diagram of interaction parameters over the temperature range of experimental data.
