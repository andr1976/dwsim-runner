UNIFAC Interaction Parameter Regression

Introduction

This tool is used to calculate missing group-group interaction parameters of the following property packages:
  1. UNIFAC
  2. Modified UNIFAC (Dortmund)
  3. Modified UNIFAC (NIST)

Loading / Saving / Creating

All informations defined in that tool, like experimental data, may be saved to a UNIFAC IP Regression study file (*.dwruf) for later use. To save this study file you call command "Save active simulation" from main menue of DWSIM.
The main menue of DWSIM also holds a command to create a new study.

Limitations

This tool is still in development. Therefore it's capabilities are still limited:
  1. Interaction parameters can only be calculated for existing groups.
  2. No new groups can be created yet.
  3. Only Liquid/Liquid equilibria data can be digested to calculate interaction parameters
  4. Calculated interaction parameters can be saved to a user database for utilisation in DWSIM simulations.
