Pump

The pump is used to provide energy to a liquid stream in the form of pressure. The process is isenthalpic, and the non-idealities are considered according to the pump efficiency, which is defined by the user

Input Parameters

Calculation mode: Select one of the following modes. Depending on selected mode the appropriate specification parameter will be displayed.
  • Delta_P: specify required pressure difference from inlet to outlet.
  • OutletPressure: specify outlet pressure.
  • Energy Stream: Energy from connected stream will be used for calculation.
  • Curves: Pump curves are specified via a special dialog.
    ==>> Pump Curve Editor

Efficiency: pump adiabatic efficiency

Ignore vapor in the input stream: defines if the calculator should ignore any vapor in the inlet stream

Output Parameters

Delta-T: temperature variation in the pumping process.

Power required: power required by the pump.

NPSH availabe: Available NPSH of pump.

Pump curves: In case of curve calculation mode the selected properties are calculated from definition tables.
