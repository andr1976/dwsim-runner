Global Settings

This tool is used to edit global settings of DWSIM

Flowsheet Options

Cut/Copy/Paste Flowsheet Objects


Defines how to handle compounds and property packages during copy/paste operations between different simulations.

Compounds: choose between not copying or copying and associating non-added compounds.

Property Packages: choose between using the default Property Package from the current simulation or copying and associating the ones from the previous simulation.

Undo/Redo


Recalculate flowsheet when undoing/redoing actions: check this box to enable flowsheet recalculation after a n undo/redo operation.

Flowsheet Object Editors


Enable multiple editors: check this box to allow multiple editor windows opened at the same time.

Close editors on deselecting: check this box to close the editor once the associated object is deselected.

Default initial placement: select the default position for the editor window to be shown at.
