Pure Component Properties Utility

The Pure Component Properties utility is used to view and edit properties of pure components. This are view molecular properties and general temperature dependent properties like ideal gas Cp, vapor pressure, liquid viscosity and vaporization enthalpy.


Editing of parameters

When you add a new component to your simulation all information upon that component are copied to your simulation file from it's original database.
If you need to modify parameters you could do this by checking the "Enable Constant Property Editing" box. Parameters which are available for editing are colored in red.

To keep track of modified components, it's status is displayed as either:
  •   Original Data  
  •   Data modified  

Your modified component parameters, together with modification status, is maintained on saving and reloading your simulation file.
If you want to restore parameters from original database you may use the button "Restore Defaults". You also can do that if you updated informations of a component in a user database and want to synchronize your simulation now.







The temperature dependent properties of each phase are listed in a separate data table.
