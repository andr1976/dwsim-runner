Database Manager

This tool is used to modify user databases

Select from dropdown list the user database which is to be modified. Only databases which are registered in "General Settings" are listed here.

Component table

This table lists all components included in the selected database. You can see component ID, name and assigned definition file ( == Component Creator study file).
If no definition file is listed here, you need to open this file again and store it into database again.

Commands


Delete Component
This will delete the selected component from database.

Edit Component
This will open the associated definition file with Component Creator tool for editing. After saving your modified component to the user database you need to import this modifications into your simulation from Pure Component Properties utility again.
