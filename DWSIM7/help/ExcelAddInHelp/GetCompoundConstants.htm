GetCompoundConstants

Returns a list of compound constants and their values from the corresponding loaded database.

Input Parameters

  • Compound name - select from component list returned by function GetCompoundList

Result

Result values are in SI units (K; Pa, m, s, ...)
The result is delivered as a two dimensional matrix with 2 columns and 18 rows.

The matrix is structured as follows:

Property Name
Property value


Example:

Parameters: yellow / Results: grey

Compound name water

molecularweight 18.015
criticaltemperature 647.14
criticalpressure 22064000
.... ....
