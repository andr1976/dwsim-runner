GetPropList

Returns a list of available single phase property names.

Input Parameters

  • Proppack - select from model list returned by function GetPropPackList

Result

The result is delivered as a one dimensional matrix containing names of available properties.

The matrix is structured as follows:

Property Name 1

Property Name 2

...

Example:

Parameters: yellow / Results: grey
