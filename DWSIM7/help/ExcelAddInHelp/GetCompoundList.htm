GetCompoundList

Returns a list of available components.

Input Parameters

  • PropPack: - select from model list returned by function GetPropPackList

Result

The result is delivered as a one dimensional matrix containing names of the components.

The matrix is structured as follows:

Component Name 1

Component Name 2

...

Example:

Parameters: yellow / Results: grey

Proppack NRTL

Air
Argon
Bromine
....
