CalcProp

Retrieves single-phase non-constant physical property values of a single phase from a mixture using the selected property package. A PT-flash is executed with given conditions and mixture composition to calculate properties of resulting phases.

Input Parameters

  • Proppack: select from model list returned by function GetPropPackList
  • Property: select from property names list returned by function GetPropList
  • Basis: Basis of the results. Valid settings are: "Mass" for Physical Properties per unit mass or "Mole" for molar properties.
    Use UNDEFINED as a place holder for a Physical Property for which basis does not apply.
  • Phaselabel: select from list of available phases returned by function GetPhaseList
  • Compounds: A list of component names in the mixture as a one dimensional matrix.
    This is usually defined as a reference to a cell range like "A1:A5" where you list the component names.
  • Temperature: Temperature in [K]
  • Pressure: Pressure in [Pa]
  • Molefractions: A list of component mole fractions in the global mixture as a one dimensional matrix.
    This is usually defined as a reference to a cell range like "B1:B5" where you list the mixture composition.
    The sum of all fractions always has to be 1.
  • IP1 ... IP8: Interaction parameter sets #1 ... #8. This allows to override standard parameters for equilibrium calculation.

Result

The result is delivered as a one dimensional matrix containing the calculated property value. The unit of the result value is always in SI units.

The matrix is structured as follows:

Property value
