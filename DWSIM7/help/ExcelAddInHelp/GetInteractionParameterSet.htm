GetInteractionParameterSet

Returns the interaction parameters stored in DWSIM's database for a given model and a binary combination of components.

Input Parameters

  • Model name - select from model list returned by function GetModelList
  • Component 1 - select from component list returned by function GetCompoundList
  • Component 2 - select from component list returned by function GetCompoundList

Result

The result is delivered as a two dimensional matrix containing the interaction parameters of the selected model.

The matrix is structured as follows:
ID1
ID2
kij / A12
kji / A21
B12
B21
C12
C21
alpha12
Component name 1
Component name 2
