GetCompoundProp

Returns a pure component property value.

For a constant property (i.e. independent of T and P like molecular weight), set T = 0 and P = 0.
For a T-dependent property, set P = 0. For a P-dependent property, set T = 0.

Input Parameters

  • Compound name - select from component list returned by function GetCompoundList
  • Property identifier - select from property list returned by function GetPopList
  • Temperature [K]
  • Pressure [Pa]

Result

Result values are in SI units (K; Pa, m, s, ...)
The result is delivered as a single value with the calculated property value.
