User database

You also can save regressed parameters to a user database. Upon starting DWSIM the next time these parameters are loaded from that database and merged into the existing interaction parameter set of that property package. If these interaction parameters already existed in the original model the new ones will replace the original ones.
BE VERY CAREFUL WHAT YOU DO HERE !!!

New: This button will allow you to create a new and empty user database for storing interaction parameters.
This database needs to be added to the interaction database list in global settings dialog.

Save: This button saves the regressed interaction parameters to the selected database. If these interaction parameters already exist they will be replaced on saving.

Search: This button will let you select an already existing database where regressed parameters will be saved to. User databases do have the file format "*.xml" and may be inspected with an internet browser.
